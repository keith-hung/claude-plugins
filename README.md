# claude-plugins

A curated Claude Code plugin marketplace.

## Installation

```bash
/plugin marketplace add claude-plugins keith-hung/claude-plugins
```

## Available Plugins

| Plugin | Description | Source |
|--------|-------------|--------|
| `read-email` | Parse .eml/.msg email files with token-optimized content extraction and attachment handling | [claude-read-email](https://github.com/keith-hung/claude-read-email) |

### Install a plugin

```bash
/plugin install read-email@claude-plugins
```
